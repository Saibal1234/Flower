# 🌸 A Flower For You

A small interactive animation that draws a **beautiful fractal flower tree** with a special message at the end.

This project was originally inspired by a **Python Turtle fractal tree**, and later recreated using **HTML Canvas and JavaScript** so it can be shared through a web link.

When the page opens, a tree slowly grows from the bottom of the screen and blossoms into a flower-like shape. After the tree finishes drawing, a special message appears.

## 🌿 How It Works

The tree is generated using **recursion**:

1. A branch grows forward.
2. The branch splits into smaller branches.
3. The process repeats until the branch length becomes very small.
4. Flower blossoms are drawn at the branch tips.

This creates a natural **fractal tree structure**.
